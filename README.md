# Introduction

Closer is a protocol that brings digital infrastructure (including web3 primitives) to community driven  land projects.\
\
This is our first draft of public documentation, so please bear with us, and for any support you hop into our [Discord](https://discord.gg/gxYsju9e) or tune in on [Telegram](https://t.me/closerearth). What would you like to see here?



Closer is composed of:

* A UI toolkit, currently a monorepo containing each of the projects.
* An API, providing standardized REST routes on a collection of models
* Smart contracts
